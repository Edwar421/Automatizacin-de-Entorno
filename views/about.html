ℹ️ Acerca de la Aplicación

Esta aplicación fue desarrollada como parte de una Kata DevOps para demostrar el uso de:

  • Node.js y Express.js para el backend
  • Docker para contenerización
  • GitHub Actions para CI/CD
  • Terraform para IaC

El objetivo es automatizar el despliegue de una app web sencilla con herramientas modernas.
